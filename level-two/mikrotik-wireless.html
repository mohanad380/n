إدارة الشبكات اللاسلكية في مايكروتك

مقدمة في الشبكات اللاسلكية في مايكروتك

تعتبر أجهزة مايكروتك من الحلول المتميزة في مجال إدارة الشبكات اللاسلكية، حيث توفر مجموعة واسعة من الميزات والإمكانيات التي تناسب مختلف احتياجات الشبكات، من الشبكات المنزلية الصغيرة إلى شبكات مزودي خدمة الإنترنت الكبيرة. في هذا الدرس، سنتعرف على كيفية إعداد وتكوين وإدارة الشبكات اللاسلكية باستخدام أجهزة مايكروتك.

واجهة إدارة الشبكات اللاسلكية في RouterOS

أنواع أجهزة مايكروتك اللاسلكية

توفر شركة مايكروتك مجموعة متنوعة من الأجهزة اللاسلكية التي تناسب مختلف الاحتياجات:

1. أجهزة hAP (home Access Point)

سلسلة أجهزة مصممة للاستخدام المنزلي والمكاتب الصغيرة، وتتميز بسهولة الإعداد والاستخدام.

  • hAP lite: جهاز بسيط وفعال من حيث التكلفة
  • hAP ac: يدعم معيار 802.11ac ويوفر أداءً أفضل
  • hAP ac²: يدعم تقنية MU-MIMO ويوفر تغطية أوسع

2. أجهزة wAP (wireless Access Point)

سلسلة أجهزة مصممة للاستخدام الخارجي، وتتميز بمقاومتها للعوامل الجوية.

  • wAP: نقطة وصول لاسلكية بسيطة للاستخدام الخارجي
  • wAP ac: يدعم معيار 802.11ac ويوفر أداءً أفضل

3. أجهزة SXT و LHG و Disc

سلسلة أجهزة مصممة للاتصالات اللاسلكية طويلة المدى (Point-to-Point و Point-to-Multipoint).

  • SXT: أجهزة ذات هوائي مدمج بكسب متوسط
  • LHG: أجهزة ذات هوائي مدمج عالي الكسب
  • Disc: أجهزة ذات هوائي قرصي للاتصالات متوسطة المدى

أنواع مختلفة من أجهزة مايكروتك اللاسلكية

إعداد نقطة وصول لاسلكية (Wireless Access Point)

يمكن إعداد جهاز مايكروتك كنقطة وصول لاسلكية بسهولة باستخدام واجهة WinBox أو من خلال سطر الأوامر (CLI).

الإعداد باستخدام واجهة WinBox

  1. افتح واجهة WinBox واتصل بجهاز مايكروتك
  2. انتقل إلى قائمة Wireless
  3. انقر على علامة التبويب Interfaces لعرض واجهات الشبكة اللاسلكية المتاحة
  4. انقر نقرًا مزدوجًا على الواجهة اللاسلكية (مثل wlan1) لفتح نافذة الإعدادات
  5. في علامة التبويب Wireless، قم بتكوين الإعدادات الأساسية:
    • Mode: ap bridge (لنقطة الوصول)
    • Band: 2GHz-B/G/N أو 5GHz-A/N/AC حسب الاحتياج
    • Channel Width: 20MHz أو 20/40MHz أو 20/40/80MHz حسب الاحتياج
    • Frequency: اختر تردد مناسب أو استخدم Auto
    • SSID: أدخل اسم الشبكة اللاسلكية
  6. انتقل إلى علامة التبويب Security Profile وقم بإنشاء ملف أمان جديد:
    • انقر على زر ... بجانب Security Profile
    • انقر على زر + لإنشاء ملف أمان جديد
    • أدخل اسمًا للملف (مثل "my_security")
    • اختر Authentication Types: WPA2 PSK
    • اختر Encryption: aes ccm
    • أدخل WPA2 Pre-Shared Key: كلمة مرور قوية
    • انقر على OK للحفظ
  7. عد إلى نافذة إعدادات الواجهة اللاسلكية واختر ملف الأمان الذي أنشأته
  8. انقر على OK لحفظ الإعدادات وتفعيل نقطة الوصول

الإعداد باستخدام سطر الأوامر (CLI)


/interface wireless
set [ find default-name=wlan1 ] mode=ap-bridge ssid=MyNetwork frequency=auto band=2ghz-b/g/n
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=my_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=my_security

إعداد جسر لاسلكي (Wireless Bridge)

الجسر اللاسلكي يستخدم لربط شبكتين منفصلتين لاسلكيًا، ويمكن إعداده بطريقتين:

1. الجسر اللاسلكي نقطة إلى نقطة (Point-to-Point)

يستخدم لربط موقعين متباعدين، مثل مبنيين.

على الجهاز الأول (نقطة الوصول):


/interface wireless
set [ find default-name=wlan1 ] mode=bridge ssid=Bridge1 frequency=5500 band=5ghz-a/n/ac
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=bridge_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=bridge_security

على الجهاز الثاني (العميل):


/interface wireless
set [ find default-name=wlan1 ] mode=station-bridge ssid=Bridge1 frequency=5500 band=5ghz-a/n/ac
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=bridge_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=bridge_security

2. الجسر اللاسلكي نقطة إلى متعدد النقاط (Point-to-Multipoint)

يستخدم لربط موقع مركزي بعدة مواقع متباعدة.

على الجهاز المركزي (نقطة الوصول):


/interface wireless
set [ find default-name=wlan1 ] mode=bridge ssid=MainBridge frequency=5500 band=5ghz-a/n/ac
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=bridge_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=bridge_security

على الأجهزة الطرفية (العملاء):


/interface wireless
set [ find default-name=wlan1 ] mode=station-bridge ssid=MainBridge frequency=5500 band=5ghz-a/n/ac
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=bridge_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=bridge_security

إعداد جسر لاسلكي في RouterOS

تقنيات متقدمة في الشبكات اللاسلكية

1. تقنية CAPsMAN (Controlled Access Point system Manager)

تقنية CAPsMAN هي نظام إدارة مركزي لنقاط الوصول اللاسلكية في RouterOS، ويسمح بإدارة عدة نقاط وصول من خلال واجهة موحدة.

لإعداد CAPsMAN على الجهاز المركزي:


/caps-man manager
set enabled=yes
/caps-man security
add authentication-types=wpa2-psk encryption=aes-ccm name=security1 passphrase=StrongPassword
/caps-man configuration
add name=cfg1 mode=ap security=security1 ssid=MyCAPsMAN
/caps-man datapath
add name=datapath1 bridge=bridge1
/caps-man channel
add band=2ghz-b/g/n name=channel1
/caps-man provisioning
add action=create-dynamic-enabled master-configuration=cfg1 name-format=prefix-identity name-prefix=CAP

لإعداد CAP (Controlled Access Point) على نقاط الوصول:


/caps-man cap
set enabled=yes discovery-interfaces=ether1

2. تقنية Wireless Distribution System (WDS)

تقنية WDS تسمح بتوسيع نطاق الشبكة اللاسلكية باستخدام نقاط وصول متعددة دون الحاجة إلى اتصال سلكي بينها.

لإعداد WDS على نقطة الوصول الرئيسية:


/interface wireless
set [ find default-name=wlan1 ] mode=ap-bridge ssid=MyWDS frequency=auto band=2ghz-b/g/n wds-mode=dynamic
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=wds_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=wds_security

لإعداد WDS على نقاط الوصول الثانوية:


/interface wireless
set [ find default-name=wlan1 ] mode=station-wds ssid=MyWDS frequency=auto band=2ghz-b/g/n wds-mode=dynamic
/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=wds_security wpa2-pre-shared-key=StrongPassword
/interface wireless
set [ find default-name=wlan1 ] security-profile=wds_security

3. تقنية Nv2 (Nstreme version 2)

تقنية Nv2 هي بروتوكول لاسلكي خاص بمايكروتك يوفر أداءً أفضل من البروتوكولات القياسية في بيئات معينة.

لتفعيل تقنية Nv2:


/interface wireless
set [ find default-name=wlan1 ] wireless-protocol=nv2 ssid=MyNv2Network

تقنيات متقدمة في الشبكات اللاسلكية في RouterOS

أمان الشبكات اللاسلكية

يوفر RouterOS العديد من الميزات لتأمين الشبكات اللاسلكية:

1. تشفير البيانات

استخدم دائمًا تشفير WPA2 أو WPA3 مع خوارزمية AES-CCM:


/interface wireless security-profiles
add authentication-types=wpa2-psk mode=dynamic-keys name=secure_profile wpa2-pre-shared-key=StrongPassword

2. إخفاء SSID

يمكن إخفاء اسم الشبكة (SSID) لتقليل ظهورها للمستخدمين غير المصرح لهم:


/interface wireless
set [ find default-name=wlan1 ] hide-ssid=yes

3. تقييد الوصول بناءً على عنوان MAC

يمكن تقييد الوصول إلى الشبكة اللاسلكية بناءً على عناوين MAC للأجهزة:


/interface wireless access-list
add interface=wlan1 mac-address=00:11:22:33:44:55 authentication=yes forwarding=yes
add interface=wlan1 mac-address=00:00:00:00:00:00 authentication=no forwarding=no

4. تحديد قوة الإشارة

يمكن تحديد قوة الإشارة لتقليل انتشارها خارج المنطقة المطلوبة:


/interface wireless
set [ find default-name=wlan1 ] tx-power=10

إعدادات أمان الشبكات اللاسلكية في RouterOS

مراقبة وتشخيص مشكلات الشبكات اللاسلكية

يوفر RouterOS العديد من الأدوات لمراقبة وتشخيص مشكلات الشبكات اللاسلكية:

1. مراقبة العملاء المتصلين


/interface wireless registration-table print

2. فحص الشبكات اللاسلكية المحيطة


/interface wireless scan wlan1

3. مراقبة استخدام النطاق الترددي


/interface wireless monitor wlan1

4. اختبار سرعة الاتصال اللاسلكي


/tool bandwidth-test 192.168.1.2

أدوات مراقبة الشبكات اللاسلكية في RouterOS

أفضل الممارسات للشبكات اللاسلكية في RouterOS

  • استخدم تردد 5 جيجاهرتز للحصول على أداء أفضل في المناطق المزدحمة
  • استخدم قنوات غير متداخلة لتقليل التداخل
  • استخدم تشفير WPA2 أو WPA3 مع كلمات مرور قوية
  • قم بتحديث البرامج الثابتة (Firmware) بانتظام
  • استخدم CAPsMAN لإدارة شبكات لاسلكية كبيرة
  • قم بإجراء مسح للشبكات المحيطة قبل اختيار القناة
  • استخدم هوائيات موجهة للاتصالات طويلة المدى
  • قم بتوثيق جميع إعدادات الشبكة اللاسلكية

خلاصة

توفر أجهزة مايكروتك مجموعة واسعة من الميزات والإمكانيات لإدارة الشبكات اللاسلكية، من الشبكات المنزلية البسيطة إلى شبكات مزودي خدمة الإنترنت المعقدة. من خلال فهم المفاهيم والتقنيات المختلفة وكيفية تطبيقها في RouterOS، يمكنك تصميم وتنفيذ حلول لاسلكية فعالة وموثوقة.

في الدرس القادم، سنتعرف على كيفية إعداد وتكوين خدمات الشبكة المختلفة على أجهزة مايكروتك، مثل DHCP و DNS و NTP.
